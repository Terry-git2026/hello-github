## hello-github

这是我的第一个 GitHub 仓库，也是一个纯静态的前端作品页（不需要 Node.js，不需要打包）。

### 在线预览（GitHub Pages）

启用 GitHub Pages 后，这个站点会发布到：

- `https://terry-git2026.github.io/hello-github/`

启用方法：仓库页面 `Settings` -> `Pages` -> `Build and deployment` -> `Source` 选择 `Deploy from a branch`，Branch 选 `main`，Folder 选 `/ (root)`，保存即可。

### 本地预览

- 直接用浏览器打开 `index.html` 就能看效果。

### 文件结构

- `index.html`：页面主体
- `styles.css`：样式
- `script.js`：交互脚本（复制邮箱、滚动动效等）
